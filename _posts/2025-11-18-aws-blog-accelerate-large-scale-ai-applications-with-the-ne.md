# ⚡ Accelerate large-scale AI applications with the new Amazon EC2 P6-B300 instances

> **原文链接**: [Accelerate large-scale AI applications with the new Amazon EC2 P6-B300 instances](https://aws.amazon.com/blogs/aws/accelerate-large-scale-ai-applications-with-the-new-amazon-ec2-p6-b300-instances/)
> **作者**: Veliswa Boya
> **发布日期**: 2025-11-18 22:16:16 UTC

## 📋 内容摘要

AWS宣布Amazon EC2 P6-B300实例全面可用，这是搭载NVIDIA Blackwell Ultra GPU的下一代GPU平台。与上一代相比，这些实例提供2倍的网络带宽和1.5倍的GPU内存，为大规模AI应用创建了平衡的平台。P6-B300实例具有6.4Tbps的EFA网络带宽、2.1TB的GPU内存，非常适合训练和部署大规模AI模型，尤其是采用混合专家（MoE）和多模态处理技术的模型。每个P6-B300.48xlarge实例配备8个NVIDIA B300 GPU、192个vCPU、4TB系统内存和8个3.84TB本地存储。这些实例现已在美国西部（俄勒冈州）AWS区域通过Amazon EC2容量块和Savings Plans提供。
