# 📰 Introducing AWS IoT Core Device Location integration with Amazon Sidewalk

> **原文链接**: [Introducing AWS IoT Core Device Location integration with Amazon Sidewalk](https://aws.amazon.com/blogs/aws/introducing-aws-iot-core-device-location-integration-with-amazon-sidewalk/)
> **作者**: Channy Yun (윤석찬)
> **发布日期**: 2025-11-13 19:09:57 UTC

## 📋 内容摘要

本文宣布AWS IoT Core设备位置服务与Amazon Sidewalk的新集成功能。该功能消除了在Sidewalk设备中安装GPS模块的需求，简化了开发者解析位置数据的体验。IoT设备制造商和解决方案开发者可以通过将蓝牙低能耗(BLE)、Wi-Fi或全球导航卫星系统(GNSS)信息发送至AWS IoT进行位置解析，然后将解析后的位置数据发送到MQTT主题或AWS IoT规则，并路由到其他AWS服务。此功能解决了以往的架构复杂性问题，无需在设备上安装耗电且昂贵的GPS硬件，延长了产品电池寿命，从而为资产跟踪和位置感知IoT应用构建更具吸引力的解决方案。该集成现已在美国东部(弗吉尼亚北部)区域正式推出。
