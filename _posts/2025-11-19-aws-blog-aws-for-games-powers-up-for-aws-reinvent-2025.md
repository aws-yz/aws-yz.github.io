# 🤖 AWS for Games powers up for AWS re:Invent 2025

> **原文链接**: [AWS for Games powers up for AWS re:Invent 2025](https://aws.amazon.com/blogs/gametech/aws-for-games-powers-up-for-aws-reinvent-2025/)
> **作者**: Jeff Harris
> **发布日期**: 2025-11-19 18:45:47 UTC

## 📋 内容摘要

AWS re:Invent 2025将于12月1日在拉斯维加斯开幕，为期五天的会议将为游戏开发者、发行商和玩家提供最新的AWS游戏技术进展。活动亮点包括：在威尼斯人博览会展厅的Agentic Arcade展示，展示由AWS for Games支持的AI辅助游戏开发；多场专业网络活动，包括游戏开发运维会议、赌博和游戏日、娱乐业AI会议和万智牌体验；丰富的技术会议，涵盖Amazon Game Studios的游戏数据基础设施、AI驱动的3D资产创建、实时AI推理模式、自主化游戏运营、Bandai Namco的元宇宙平台、Rovio的生成式AI资产创建以及EA的Apex Legends迁移到Amazon GameLift等话题。与会者可通过AWS Events应用程序获取最新更新和安排行程。
