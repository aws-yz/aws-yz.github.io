# 📰 Monitor network performance and traffic across your EKS clusters with Container Network Observability

> **原文链接**: [Monitor network performance and traffic across your EKS clusters with Container Network Observability](https://aws.amazon.com/blogs/aws/monitor-network-performance-and-traffic-across-your-eks-clusters-with-container-network-observability/)
> **作者**: Donnie Prakoso
> **发布日期**: 2025-11-19 19:05:59 UTC

## 📋 内容摘要

AWS宣布推出Amazon EKS的容器网络可观察性功能，这是一套全面的网络可观察性工具，帮助组织更好地监控Kubernetes集群中的网络性能和流量模式。该功能提供三个主要组件：服务地图(Service Map)用于可视化工作负载间的通信；流量表(Flow Table)提供AWS服务视图、集群视图和外部视图三种视角来监控网络流量；性能指标端点收集网络相关系统指标。通过AWS Network Flow Monitor Agent，用户可以获取详细的网络流量分析，识别性能问题，并优化应用程序拓扑。该功能可与Prometheus和Grafana兼容，适用于所有提供Amazon CloudWatch Network Flow Monitor的商业AWS区域，按标准CloudWatch Network Flow Monitor定价收费。
