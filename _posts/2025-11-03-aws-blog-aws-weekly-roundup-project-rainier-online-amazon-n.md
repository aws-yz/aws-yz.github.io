# ⚡ AWS Weekly Roundup: Project Rainier online, Amazon Nova Web Grounding, MCP Proxy for AWS, and more (November 3, 2025)

> **原文链接**: [AWS Weekly Roundup: Project Rainier online, Amazon Nova Web Grounding, MCP Proxy for AWS, and more (November 3, 2025)](https://aws.amazon.com/blogs/aws/aws-weekly-roundup-project-rainier-online-amazon-nova-amazon-bedrock-and-more-november-3-2025/)
> **作者**: Betty Zheng (郑予彬)
> **发布日期**: 2025-11-03 17:55:23 UTC

## 📋 内容摘要

本周AWS重点更新：AWS与Anthropic合作的Project Rainier超级计算机上线，集成近50万颗Trainium2芯片；Amazon Nova新增Web Grounding和多模态嵌入功能；MCP Proxy正式发布，支持AWS远程MCP服务器连接；Amazon ECS支持内置线性和金丝雀部署策略；Amazon DocumentDB推出新查询规划器，查询性能提高10倍；Amazon EBS发布新的每卷CloudWatch指标；Kinesis Data Streams支持10倍更大的记录大小(10MiB)。其他更新包括AWS VAMS和4D Pipeline的3D资产管理解决方案、Amazon Location Service新增API密钥限制、AWS Clean Rooms推出高级SQL配置等。近期活动包括AWS Builder Loft、全球多地AWS社区日以及西雅图AWS技能中心四周年庆祝活动。
