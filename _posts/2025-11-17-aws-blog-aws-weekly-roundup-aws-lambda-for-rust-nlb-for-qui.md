# 📰 AWS Weekly Roundup: AWS Lambda for Rust, NLB for QUIC protocol, Amazon DCV for Mac, and more (November 17, 2025)

> **原文链接**: [AWS Weekly Roundup: AWS Lambda for Rust, NLB for QUIC protocol, Amazon DCV for Mac, and more (November 17, 2025)](https://aws.amazon.com/blogs/aws/aws-weekly-roundup-aws-lambda-load-balancers-amazon-dcv-amazon-linux-2023-and-more-november-17-2025/)
> **作者**: Sébastien Stormacq
> **发布日期**: 2025-11-17 17:59:08 UTC

## 📋 内容摘要

本周AWS更新包括：AWS Lambda正式支持Rust语言、Java 25以及Swift实验性运行时；Lambda为SQS事件源映射增加了预置模式；EventBridge推出增强型可视化规则构建器；AWS服务参考信息现支持SDK操作到IAM操作的映射；网络负载均衡器(NLB)支持QUIC协议直通模式，可降低25-30%的应用延迟；应用负载均衡器(ALB)支持JWT验证的客户端凭证流；AWS KMS支持Edwards曲线数字签名算法；Amazon DCV支持EC2 Mac实例；Amazon Linux 2023发布新版本。此外，博客还提及了AWS生成式AI大厅在以色列特拉维夫开放，AWS Heroes项目欢迎三位新成员，以及即将到来的AWS社区活动，包括AWS Builder Loft、AWS社区日和AWS技能中心西雅图四周年庆典。
