# 🗄️ Hypermonk Games transforms mobile game development with data powered by AWS

> **原文链接**: [Hypermonk Games transforms mobile game development with data powered by AWS](https://aws.amazon.com/blogs/gametech/hypermonk-games-transforms-mobile-game-development-with-data-powered-by-aws/)
> **作者**: Didem Koskos Gurel
> **发布日期**: 2025-10-28 18:02:25 UTC

## 📋 内容摘要

土耳其游戏工作室Hypermonk Games通过构建名为Orange的数据分析基础设施，彻底改变了其移动游戏开发方式。该基础设施是与AWS和AWS合作伙伴Clerion共同开发的，使用了AWS Lambda、Amazon Kinesis、Amazon DynamoDB、Amazon S3、Amazon SQS和Amazon EventBridge等AWS服务。Orange提供了实时玩家数据分析、广告收入跟踪和用户获取数据整合，帮助团队优化投资回报率并做出数据驱动的决策。与传统的分散式第三方工具相比，Orange集中化的方法提高了开发效率，降低了成本（存储成本降低约60%，数据库运营成本降低约50%），并使团队能够专注于产品开发和创新。Hypermonk Games计划未来进一步发展Orange，增加AI和机器学习功能，以简化仪表板使用并提升数据分析洞察力。
