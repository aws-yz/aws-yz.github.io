# 🤖 MSD explores applying generative Al to improve the deviation management process using AWS services

> **原文链接**: [MSD explores applying generative Al to improve the deviation management process using AWS services](https://aws.amazon.com/blogs/machine-learning/msd-explores-applying-generative-al-to-improve-the-deviation-management-process-using-aws-services/)
> **作者**: Hossein Salami, Jwalant (JD) Vyas
> **发布日期**: 2025-11-20 18:21:49 UTC

## 📋 内容摘要

这篇博客文章介绍了默克公司（MSD）如何利用生成式AI和AWS服务优化制药生产过程中的偏差管理。MSD的数字制造数据科学团队开发了一个创新解决方案，使用Amazon Bedrock和Amazon OpenSearch服务，将历史偏差报告转化为可查询的知识库。该解决方案采用检索增强生成（RAG）架构，结合语义和关键词搜索实现高准确度的信息检索。系统使用Amazon RDS存储结构化数据，通过OpenSearch索引过去偏差报告的嵌入向量，并通过Amazon Bedrock托管的大型语言模型生成见解。这一创新旨在显著减少偏差调查所需的时间和精力，同时确保符合GMP标准。未来，MSD计划将这一解决方案扩展到更多制药质量领域，并考虑利用Amazon Bedrock Knowledge Bases提供更先进的语义搜索和检索功能。
