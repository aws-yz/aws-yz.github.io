# 🤖 Build more accurate AI applications with Amazon Nova Web Grounding

> **原文链接**: [Build more accurate AI applications with Amazon Nova Web Grounding](https://aws.amazon.com/blogs/aws/build-more-accurate-ai-applications-with-amazon-nova-web-grounding/)
> **作者**: Matheus Guimaraes
> **发布日期**: 2025-10-28 23:59:17 UTC

## 📋 内容摘要

AWS宣布Amazon Bedrock上的Nova模型现已正式推出Web Grounding功能。这是一种内置工具，为开发者提供即用型检索增强生成(RAG)选项，使Nova基础模型能根据提示上下文智能决定何时检索并融合相关的最新信息，通过引用公共资源作为上下文来减少幻觉并提高准确性。Web Grounding特别适用于需要获取最新事实信息或提供有充分引用的响应的应用程序，如知识型聊天助手、内容生成工具、研究助手和客户支持应用程序。该功能现已在美国东部(弗吉尼亚北部)、美国东部(俄亥俄州)和美国西部(俄勒冈州)推出，目前仅支持Nova Premier模型，使用时会产生额外费用。
