# 💾 emptyvessel builds global game development pipeline with AWS

> **原文链接**: [emptyvessel builds global game development pipeline with AWS](https://aws.amazon.com/blogs/gametech/emptyvessel-builds-global-game-development-pipeline-with-aws/)
> **作者**: Lilly McDermott
> **发布日期**: 2025-10-23 17:03:02 UTC

## 📋 内容摘要

这篇博客介绍了游戏开发工作室emptyvessel如何利用AWS服务构建全球分布式游戏开发流程。emptyvessel成立于2023年，是一家远程优先的工作室，团队成员分布在从悉尼到斯洛伐克的全球各地，正在开发名为DEFECT的赛博朋克射击游戏。为解决全球协作和快速开发周期的挑战，emptyvessel采用了基于云的开发管道，包括使用Amazon EC2托管Perforce数据库，Amazon VPC优化网络架构，AWS Global Accelerator确保全球访问速度，以及EventBridge驱动的自动化构建系统。他们还实施了Amazon S3的存储优化策略，利用DynamoDB管理游戏数据，并通过AWS Lambda和Amazon ECS实现无服务器架构。这一基础设施使团队能够专注于游戏开发而非基础设施管理，显著提高了开发效率，减少了故障排查时间，并为游戏的未来扩展提供了可扩展性支持。
